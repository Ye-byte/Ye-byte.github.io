Live Server

启动具有静态和动态页面实时重新加载功能的本地开发服务器

可以将修改的html文件实时展示,去掉每次修改html后都要重新拖入浏览器的麻烦.

安装

  • 打开vs code左侧的插件图标
  • 插件栏中搜索’Live Server’,点击install安装
  • 安装成功后,右下角即会出现Go Live的字体

使用Live Server

两种使用方法:

  • 右键html文件’open with Liver Server’
  • 打开HTML文件,右下角Go Live执行

!!!注意:修改后一定要保存以后才能生效
