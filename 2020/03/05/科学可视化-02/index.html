感知与认知

视觉感知和认知

感知:
指客观事物通过人的感觉器官在人脑中形成的直接反应
感觉器官:

认知:
指人们获得知识或应用知识的过程,或信息加工的过程. eg:就是人们如何理解看到的事物,例如看到一只狗,知道这是一只狗,凭多年的经验判断而不是一只猫.
三个结论:
结论一: 记忆在人类认知过程中期至关重要的作用,但工作容量十分有限.
结论二:观察物体的变化需要集中注意力.在可视化中突出变化,可以减少认知负担
结论三:感知系统基于相对判断,而非绝对判断.使用相同的参照物或者相互对齐,有助于人们做出更加准确的相对判断.

格式塔理论

结构比元素重要,视觉形象首先作为统一的整体认知
将事物看做一个整体,感知内容作为常规的简单的结构
定义八大原则:

  • 接近原则
    当视觉元素在空间距离上相距较近时,人们通常倾向于将它们归为一组

  • 相似原则
    • 人们在观察事物的时候,会自然地根据事物的相似性进行感知分组
    • 通常依据对形状 颜色 光照或者其他性质的感知绝对分组

  • 连续原则
    人们在观察事物的时候会很自然到地沿着物体的边界,将不连续的物体视为连续的整体
  • 闭合原则
    • 在某些视觉映像中,其中的物体可能是不完整的或者不闭合的
    • 只要物体的形状足以表征物体本身,人们会很容易感知整个物体二忽视未闭合的特征.

  • 共势原则
    一组物体具有沿着相似的光滑路径运动趋势或者具有相似的排列时,将被视为同一类物体.
  • 好图原则
    人眼通常会消除复杂性和不熟悉性来理解被识别的物体.
  • 对称原则
    人的意识倾向于将物体识别为沿某点或某轴对称的形状

  • 经验原则
    某些情形下视觉感知与过去的经验有关.如果两个物体看上去距离相近,或者时间间隔小,那么它们通常会被识别为同一类.

    基本思想

    视觉形象首先是作为统一的整体被认知的,而后才以部分的形式被认知.
    先看见图的整体,才看到构成图的部分

    练习

1.相近原则 2.连续原则 3.相似原则

    我可以选用什么视觉通道来进行编码?

    从下面几个方面入手:
  • 首先看数据类型,数据有如下三种
    • 类别型 -> 区分事物
      例如性别
    • 有序型 -> 表示对象间的顺序结构
      衣服的尺码
    • 数值型 -> 表示数据的定量数值
      例如商品的价格
  • 视觉通道的选择
    • 视觉通道分类
      • 定性/分类型
        描述感知对象是什么或在哪里适合编码分类型的数据信息
        eg:
        形状 颜色色调 空间位置
      • 定量/定序型
        描述感知对象某一属性的具体数值是多少
        适合编码有序型的或者数值型的数据信息
        eg:
      • 分组型
        描述多个或多种标记的组合
        适合将存在相互联系的分类的数据属性进行分组,从而表现数据的内在关联性
        eg:
  • 表现力原则
    视觉通道的表现力要求视觉通道准确编码数据包含的所有信息.也就是说,视觉通道在对数据进行变敏感的时候,需要尽量忠于原始数据.
    eg:有序数据 -> 定序的数据通道
  • 有效性原则
    表现力更高的视觉通道编码更重要的数据信息
    表现力排序图:
  • 视觉通道表现力的标准
    • 精确性
      描述人类感知系统对于可视化的判断结果和原始数据的吻合程度
      基于史蒂文斯幂次法则判断准确性
      eg:
    • 可辩性
      如何在给定的取值范围内选择合适数目的不同取值,使得人类的感知系统可以轻易区分该视觉通道的两种或多种取值状态.
      eg:
      直线宽度 -> 面积
    • 可分离性
      描述不同视觉通道再被用于表达数据属性的时候,两两之间的干扰现象.如何在给定的取值范围
      eg:

      圆的大小影响了颜色的判断
    • 视觉突出
      在很短的时间内(200~250毫秒),仅仅依赖感知的低阶视觉即可直接察觉某一对象和其他所有对象的不同对象
      eg:

      无论绿点的多少,都可以很快找出红点
